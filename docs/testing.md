# Testes Manuais

| Caso de teste | Ação | Resultado esperado |
|----------------|------|--------------------|
| Adicionar | Preenche todos os campos e salva | Nova mídia aparece na tabela |
| Atualizar | Seleciona item e altera dados | Campos atualizados |
| Excluir | Seleciona e deleta | Item some da lista |
| Filtrar por Tipo | Seleciona “Filme” | Exibe só filmes |
| Ordenar Nota Crescente | Seleciona “Crescente” | Exibe itens com nota crescente |
| Ordenar Nota Decrescente | Seleciona “Decrescente” | Exibe itens com nota decrescente |
